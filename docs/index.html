What is Pricing Pal?

Pricing Pal is an Android application built with non-profit thrift stores in mind, although it may be useful to others too! We will offer cloud storage and quick editing of basic pricing lists, so that everyone who works in your store can have easy access to your up-to-date pricing list -- no sign-up required. You, as a registered admin user, will be able to create and edit your own pricing lists, in-app, from anywhere with an Internet collection. This project is new and still in development, so keep checking for new features!

How Do I Use Pricing Pal?

For Users

Currently, Pricing Pal is only available through downloading the APK file from our GitHub repository. Once you have the APK, you can install it on your Android device to see our user interface and basic functionality. Creation of a user account is not yet available. We hope to have the fully-featured app published to the Google Play Store and F-Droid in the near future, at which time those links will be added to the bottom of this page.

For Developers

Our GitHub repository is public, and we welcome any and all contributions to the project. If you would like to contribute, please fork the repository and submit a pull request. We will review your changes and merge them into the main branch if they are acceptable. If you have any questions, check our README.md file for technical, licensing, and contact information.
